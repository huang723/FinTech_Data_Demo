# FinTech_Data_Demo
# A股茅台股价自动化分析系统 (Maotai Stock Analysis)

##项目背景
本项目旨在利用 Python 对 A 股龙头企业（贵州茅台 600519.SH）进行金融数据获取与技术分析。作为金融硕士与本科信管复合背景的实践，探索从数据采集到可视化分析的全流程自动化。

## 技术栈
- **核心语言**: Python 3.14
- **数据接口**: Tushare Pro (金融大数据开放社区)
- **数据处理**: Pandas (数据清洗与 ETL)
- **可视化**: Matplotlib (股价走势与均线绘制)

## 实现功能
1. **自动化采集**: 调用 Tushare API 获取指定时间段的日线行情。
2. **数据清洗**: 处理时间序列索引，转换数据格式。
3. **技术分析**: 计算并绘制 **5日均线 (MA5)** 与 **30日均线 (MA30)**。
4. **可视化输出**: 自动生成高清股价走势图并导出。
